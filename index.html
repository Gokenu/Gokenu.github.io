lc4 Median of Two Sorted Arrays

Posted on 2021-01-12 In leetcode

Explanation From Others

Explanation From Others
If we can ensure:
1 len(left_part) == len(right_part)
2 max(left_part) <= min(right_part)
then we divide all elements in {A, B} into two parts with equal length, and one part is always greater than the other. Then median = (max(left_part) + min(right_part))/2.

Explanation By Myself

We can use binary search to find the index i,j where we devide A and B.

Objective:

  1. len(left_part) == len(right_part)
  2. max(left_part) <= min(right_part)

二分查找的思考

  1. 推荐使用下界的查找方式: 即找满足(func(x)==true) 的最小x的位置
  2. 使用左闭右开区间定义[left,right)
